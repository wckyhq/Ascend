# Ascend

**A free and open source reminder to use that standing desk!**

<img src="/misc/Ascend-01.png" alt="Ascend-Banner">

Ascend is a MacOS application that lives in your menu bar. You can configure time between standing/sitting, the icon on the menu bar, and the sound either state makes.

## How It Works

Ascend will remind you to stand/sit (default is 30 minute intervals). That's it! It is fully open-source, private, and very simple.

## Installation

1. Download the latest release from the [releases page](https://github.com/wckyhq/Ascend/releases).
2. Install the application like you would any other MacOS app.
3. Open System Settings, scroll to "Privacy & Security", scroll to the bottom of the page and press "Open Anyway". 
4. Launch Ascend.
5. Configure your preferred intervals, sounds, and icons.
6. Finally use that standing desk!
